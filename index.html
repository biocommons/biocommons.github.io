About

biocommons (http://biocommons.org/) is a collection of software tools related in some way to sequence analysis, and particularly genome sequence analysis. Most of the tools were originally written by a few colleagues at Invitae. We hope that you'll use and contribute to these tools.

Names like "biocommons" are, well, common. You may also wish to see the other unaffiliated sites with similar names: Bio-Commons (with a hyphen), UMN biocommons, or UW BioCommons.

Packages

bioutils [ Issues]

This package provides some simple tools and lookup tables used primarly by the hgvs and uta packages.

eutils [ Issues]

eutils is a Python package dedicated to searching for, fetching, and parsing records from NCBI using their E-utilities interface.

hgvs [ Issues]

This package provides a Python library to facilitate the use of genome, transcript, and protein variants that are represented using the Human Genome Variation Society (Mutnomen) recommendations.

multifastadb [ Issues]

MultiFastaDB presents a collection of indexed fasta files as a single source. The intent is to simplify accessing a virtual database of sequences that is distributed across multiple files.

uta [ Issues]

The UTA (Universal Transcript Archive) stores transcripts aligned to sequence references (typically genome reference assemblies). It supports aligning the same transcript to multiple references using multiple alignment methods.

uta-align [ Issues]

uta-align provides C-based Needleman-Wunsch and Smith-Waterman alignment algorithms with a Python interface.
